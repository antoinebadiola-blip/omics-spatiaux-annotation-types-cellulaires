# Annotation des types et sous-types cellulaires en transcriptomique spatiale

Ce dépôt contient le code associé à un mémoire de fin d’études portant sur l’identification des types et sous-types cellulaires à partir de données de transcriptomique spatiale.

Les analyses reposent sur des jeux de données MERFISH issus de l’Allen Brain Cell Atlas et s’appuient sur un pipeline structuré et reproductible, combinant des approches data-driven (clustering) et knowledge-driven (annotation par gènes marqueurs).

Auteur : Antoine Badiola  
Année universitaire : 2025–2026
